# Testimonial headshots

Put actor headshot images here for the landing testimonials section.

- **Path in code:** reference as `/testimonials/filename.jpg` (or `.jpeg`, `.png`).
- **Naming:** use lowercase, no spaces (e.g. `jane-doe.jpg`, `canberk.jpeg`).
- **Format:** JPG or PNG; keep file size reasonable (e.g. under 200KB) for fast loading.

Example: add `jane-doe.jpg` here, then in `LandingTestimonials.tsx` set `image: "/testimonials/jane-doe.jpg"` for that testimonial.
